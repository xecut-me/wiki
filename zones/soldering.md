# Soldering Zone

Soldering zone includes:

- [[Tools/Soldering Station]]
- [[Tools/Desoldering Station]]
- Additional tools and materials

## Controls

Dedicated remote control can be used:
- Turn the workbench on using the white button ( | symbol).
- Turn the workbench off using the white button ( O symbol).

The soldering station usualy turns on together with the workbench lighting.
Station can also be switched on and off using the power switch on the back side of it.

Flux is somewhere on the right side, and the solder is stored there as well.

280°C is a great default temperature for the soldering iron.
